# Colossi-MMORPG-Vertical-Slice
Welcome to the Colossi of the Afterlife MMORPG vertical slice! This is an early-stage prototype of a mythic, cooperative MMORPG where players live inside giant bio-mech colossi, forming societies based on organelle functions. Our vertical slice focuses on core mechanics like Warmth, Stability, and cooperative "Song &amp; Dance" events.
